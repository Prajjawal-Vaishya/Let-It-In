Let It In
"Silence is your only weapon. The AI is listening."

1. About The Project
Let It In is an immersive psychological survival game that blurs the line between player and protagonist. Unlike traditional horror games, this project utilizes real-time audio input and adaptive AI to create a responsive environment where your physical reactions dictate your survival.

Designed to test critical thinking under pressure, the game forces players to solve complex psychological puzzles while being hunted by a custom-made AI entity. It serves not only as a game but as a simulation tool for high-stress scenarios, such as defense training or escape strategies.

3. Powered by GenAI
This project leverages cutting-edge Generative AI to create a non-repetitive, organic experience:

Dialogue by Gemini: The narrative structure, lore, and dynamic dialogue options are powered by Google Gemini, ensuring that the story adapts and feels intellectually challenging.

Voices by ElevenLabs: The antagonist and environmental voices are generated using ElevenLabs, delivering hyper-realistic, emotive, and terrifyingly human-like audio that reacts to the player's presence.

4. Real-World Applications
Beyond entertainment, this engine is designed with serious simulation capabilities:

Defense & Tactical Training: Can be used as a sound system simulation for defense sectors, training personnel to maintain noise discipline in hostile environments.

Safety Simulations: serves as an educational tool for scenarios like kidnap escape, teaching users how to remain calm and quiet while seeking exit strategies in hostile confinement.
